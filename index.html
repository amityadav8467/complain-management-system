Features

Everything you need to manage grievances

From secure authentication to real-time tracking — built for speed, clarity, and accountability.

🔐
JWT Authentication

Secure login and registration with industry-standard token-based auth for users and admins alike.

📡
Real-time Tracking

Monitor complaint status live — from submission to resolution, every step is fully visible.

📎
File Attachments

Attach images, PDFs, and documents directly to your complaint for stronger supporting context.

🛠️
Admin Dashboard

Powerful admin panel with full control over complaints, users, assignments, and analytics.

👥
Role-based Access

Separate portals for users, staff, and admins — each with carefully tailored permissions.

🎯
Priority Filtering

Sort and filter by status, category, or priority level for faster and more targeted resolution.
